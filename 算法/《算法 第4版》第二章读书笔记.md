排序就是将一组对象按照某种逻辑顺序重新排列的过程。

- 自反性：对于所有的V,V=V;
- 反对称性：对于所有的 V<W 都有 V>W,且 V=W 时 W=V;
- 传递性：对于所有的V、W和X ，如果 V<=W 且 W<=X ,则 V<=X 。

# 选择排序

一种最简单的排序算法是这样的：首先，找到数组中最小的那个元素，其次将它和数组的第一个元素交换位置（如果第一个元素就是最小元素那么它就和自己交换）。再次，在剩下的元素中找到最下的元素，将它与数组的第二个元素交换位置。如此往复，直到将整个数组排序。这种方法叫做选择排序，因为它在不断地选择剩余元素之中的最小者

对于长度为N的数组，选择排序需要大约N2/2次比较和N次交换。


- 运行时间和输入无关
- 数据移动是最少的



```


	public class Selection{
	
		public static void sort(Comparable[] a){
			int N = a.length;
			for(int i = 0 ; i < N ; i++){
				int min = i;
				for(int j = i+1; j < N; j++){
					if(less(a[j],a[min])){
						min = j;
					}
					exch(a,i,min);
				}
			}
		}
	
	}
    
```
